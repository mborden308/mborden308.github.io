# Website Design Strategy: Landscape Photography Portfolio

## 1. Core Identity & Visual Language
* **Design Philosophy:** Minimalism and "Expansiveness." The UI should be invisible; the art is the hero.
* **Color Palette:** Neutral/Monochromatic (White, Light Gray, or Deep Charcoal). Avoid vibrant UI colors that compete with natural landscapes.
* **Typography:** * *Primary (Headings):* Clean Sans-Serif (e.g., Montserrat) for modernism or Elegant Serif (e.g., Playfair Display) for an editorial feel.
    * *Body:* High-readability Sans-Serif (e.g., Lato or Open Sans).
* **Whitespace:** High margins and generous padding to create a "Gallery" atmosphere.

## 2. Site Architecture
* **Home Page:** * Full-width "Cinema" Hero image (High-impact landscape).
    * Minimalist top navigation (avoid sidebars to preserve horizontal width).
* **Galleries (Categorized):**
    * **The Wilderness:** Primary outdoor landscapes.
    * **The Concrete Jungle:** Urban scapes and architecture.
    * **The Fine Details:** Bokeh and close-up/macro photography.
* **About Page:** * Professional headshot (contextual: outdoors/with gear).
    * Artist statement & philosophy.
* **Contact Page:** Simple form, social links, and clear call-to-action.

## 3. Technical Requirements & Professional Features
* **Layout Engine:** Use Masonry Grids for galleries to preserve varying aspect ratios without awkward cropping.
* **Image Optimization:**
    * **Format:** sRGB color space for web consistency.
    * **Loading:** Lazy loading enabled for speed.
    * **Lightboxes:** High-res "Click to Enlarge" functionality for detail inspection.
* **SEO:** * Descriptive file naming (e.g., `oregon-coast-sunset.jpg`).
    * Alt-text for all images.
* **Proofing:** Password-protected client galleries (if applicable for future business).

## 4. Mobile-First Optimization
* **Responsive Design:** Elements must stack vertically on mobile.
* **Focal Point Cropping:** Set manual focal points for banner images to ensure mobile crops don't lose the subject.
* **Interaction:** Large, "fat-finger" friendly buttons and navigation.
* **Rotation:** Support for landscape orientation viewing.

## 5. Content Curation Guardrails
* **Quality over Quantity:** Limit galleries to 12-15 elite images.
* **Visual Flow:** Group images by tone, color, or lighting to ensure a cohesive scroll.
* **Aspect Ratios:** Mix in vertical landscapes to improve the mobile browsing experience.
